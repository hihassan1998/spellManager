Welcome to the Spelling Manager!

This application allows you to manage a dynamic word list using a Trie data structure.

You can:

  • Search for words and prefixes efficiently.
  • Remove words, with the option to restore them later.
  • Switch between different word lists easily.
  • Clear session data to reset the dictionary.
See the About page for details.
